<details><summary> Linux </summary>

```sh
git clone https://github.com/bakrch/nvim.git "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
```

</details>

<details><summary> Windows </summary>

If you're using `powershell.exe`

```
git clone https://github.com/bakrch/nvim.git $env:USERPROFILE\AppData\Local\nvim\ 
```

</details>

### Post Installation

Start Neovim

```sh
nvim
```
